고객센터 홈으로 돌아가기

가입한 이메일 주소와 비밀번호를 변경하고 싶어요.

이메일과 비밀번호는 '설정'에서 변경할 수 있어요. 아래 방법을 확인해 주세요!
PC 웹에서 변경하기
- 이메일 변경 : 설정 > 프로필의 이메일 변경
- 비밀번호 변경 : 설정 > 프로필의 비밀번호 변경
모바일 App에서 변경하기
- 이메일 변경 : 마이페이지 > 설정 > 계정 설정 > 이메일 변경
- 비밀번호 변경 : 마이페이지 > 설정 > 계정 설정 > 비밀번호 변경
